% \begin{abstract}
% 
% 
% We study compression algorithms for Euclidean spaces with respect to center-based clustering objectives.  The strongest guarantee a compression algorithm can
% satisfy is the coreset guarantee, which states that the cost of any candidate set of centers on the coreset and original data set are equal up to $(1\pm
% \varepsilon)$ factors. Unfortunately, despite extensive efforts, computing coresets is expensive.  For this reason, a number of computationally
% simpler methods have been proposed, the simplest of which consists of uniform sampling which does not even have to read the entire dataset.
% 
% In this paper, we show how to construct a coreset for Euclidean $k$-median and $k$-means in near linear time $\tilde O (nd)$. Moreover, no sublinear time
% algorithm is possible, making this result essentially optimal with respect to the running time, while retaining the space guarantee of any offline coreset
% construction.  We implement this algorithm and illustrate its efficiency on real-life datasets.  In particular, it is faster than any known coreset algorithm
% with provable guarantees.
% \end{abstract}
